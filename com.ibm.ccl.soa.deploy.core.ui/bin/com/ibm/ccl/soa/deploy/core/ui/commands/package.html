The SOA Deploy Core Editor is extended by each domain to allow editing that domain’s model.

Deploy Core Editor “Commands” Package API Documentation

Abstract [Index]

 

Contains the editor’s command classes.

 

Command Classes [Top]

 

Command classes perform the changes to the model and can also be used to undo and redo the changes to the model. Command classes are typically invoked from the Action classes as the result of menu/toolbar actions or are created by an edit policy class that’s installed on an editpart class in response to a request class.  Request classes are created and sent to editpart classes, as an example, by the tool classes to create a new figure in the diagram.
