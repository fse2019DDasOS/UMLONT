The 'Deployment Editor' Eclipse Preferences implementation.

Deploy Core Editor “Preferences” Package API Documentation

Abstract [Index]

 

This package implements the org.eclipse.ui.preferencePages extension point. It supplies two pages to the Eclipse Preferences system, a main page and a Validation Settings page.

In this document, concepts are enclosed by italics (like this), references to types or methods in the Java API are enclosed by fixed-width fonts (like this), and references to elements in the schemas are enclosed in angle brackets(<likethis />).

 

Preferences Implementation [Top]

 

The main Deployment Editor Preferences page controls the default behavior of the Deployment Perspective and the default line styles used on the Deployment Topology Diagram.

The Validation Preferences page allows fine-grained control of the Deployment Validator.

 

The Preference Implementation will be tested by interactively:

  • Verifying that both pages appear in the Preferences dialog.
  • Verifying that the main Deployment Editor Preferences page controls correctly set the preferences.
  • Verifying that the Validation Preferences page controls the action of the validator by introducing validation errors and suppressing validation of them.
