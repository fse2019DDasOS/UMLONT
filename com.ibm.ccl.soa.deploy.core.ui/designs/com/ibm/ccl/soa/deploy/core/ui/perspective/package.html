The 'Deployment' Eclipse perspective implementation.

Deploy Core Editor “Perspective” Package API Documentation

Abstract [Index]

 

Implementation of the org.eclipse.ui.perspectives extension point for the SOA Deploy Core Editor. This creates the Deployment Perspective within the Eclipse IDE.

In this document, concepts are enclosed by italics (like this), references to types or methods in the Java API are enclosed by fixed-width fonts (like this), and references to elements in the schemas are enclosed in angle brackets(<likethis />).

 

Deployment Perspective implementation [Top]

 

This package only contains one class, com.ibm.ccl.soa.deploy.core.ui.perspective.DeployCorePerspectiveFactory, which merely implements IPerspectiveFactory, the standard org.eclipse.ui.perspectives <extension point>.

The SOA Deploy Core Editor Perspective contributes a Deployment Navigator view, a Server Navigator view, and Publish Problem view, an Export Problem view, all of which are custom to the SOA Deploy Core Editor. In addition, the standard Eclipse Outline, Property Sheet, Problem, and Task List views are contributed.

The Publish Problem and Export Problem views are private but documentation for the navigators will be found in the com.ibm.ccl.soa.deploy.core.ui.navigator.deploy package.

 

 

Testing Requirements [Top]

 

The Perspective implementation will be tested by interactively entering the Deployment perspective and locating the following views:

  • Deployment Navigator
  • Server Navigator
  • Publish Problem
  • Export Problem
