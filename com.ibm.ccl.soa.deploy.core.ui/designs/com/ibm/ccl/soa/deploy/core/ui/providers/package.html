The SOA Deploy Core Editor is extended by each domain to allow editing that domain’s model.

Contains Provider Classes

 

Provider Classes [Top]

 

Provider classes allow this editor to use GMF’s services to provide services to this editor. As an example the MarkerDecoratorProvider configures the Decorator Service to provide decorations on figures in the diagram.
