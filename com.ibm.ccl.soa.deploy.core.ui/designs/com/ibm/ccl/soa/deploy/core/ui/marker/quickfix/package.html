Deploy Core Editor “Quickfix Marker” Package API Documentation

Abstract [Index]

 

This package contains the implementation of Quick fixes in the SOA Deploy Core Editor. If a validation error has a known Resolution, the UI marks that error with a light bulb in addition to its error decoration. When the light bulb is clicked by a user, this package runs code to present a list of possible resolutions intended to correct the error.

Quickfix implementation and API [Top]

 

This package contains the user interface code to present a list of Validation Resolutions. The list appears as a box near the error marker the user clicked to indicate his or her desire to correct a problem.

Third parties wishing to add similar behavior to their own GMF EditParts beyond what is provided by the SOA Deploy Core Editor may use the QuickFixPopUp class.

Third parties wishing to add their own Validation Resolutions appearing under this UI mechanism must implement the com.ibm.ccl.soa.deploy.core.resolutionGenerators extension point.
