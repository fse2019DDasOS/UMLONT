Composite:

A Composite is an Implementation that describes a collection of Components, Services, and References. It also provides a base where WireSource/WireTargets may be resolved.

Formerly, there was a Module implmentation that contained Components configured for a specific deployment. Composite replaced Module and the Module hierarchy, as well as Subsystem.
