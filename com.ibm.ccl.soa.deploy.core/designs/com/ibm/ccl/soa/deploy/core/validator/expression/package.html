Requirement expression interpreter interfaces and core implementations.

Requirement Expression Interpreters Overview Documentation

Abstract [Index]

Requirement expressions are associated with a interpreter attribute which is used to check that the value of the expression matches the target of the requirement link target. This package contains a set of common interpreters such as Equals and LessThan. Additional interpreters can be defined using the domain extension point.
