Topology Discoverer Service

Abstract [Index]

Units can be "discovered" outside the current topology using a TopologyDiscoverer. A topology discoverer service provides a way to query any number of discoverers for units that meet the conditions defined in a discovery filter. Discovered units can then be added to the topology.

A discovery filter factory provides static methods to create filters for particular purposes and to determine if a filter is for a particular purpose.
